Multimodal Metronome and Conducting Tutor

NOTE: In order to use the application, run the ‘main.py’ file from inside the ‘src’ folder 


OS: Windows, Language: Python2

Hardware: Leap Motion Controller

Installations:

Package	 | Version | Comments/Installation
--------------------------------------------------
Leap SDK | V3.2    | Included in ‘lib’ folder
--------------------------------------------------
Pillow   |6.0.0    | Used for displaying images
         |         | pip install pillow
--------------------------------------------------
pyttsx	 |1.1      | Python text-to-speech module
         |         | pip install pyttsx
--------------------------------------------------
Winsound |1.5.2    | Provides access to Windows 
         |         | sound-playing machinery.
--------------------------------------------------
pypiwin32|         | Python extension for Windows. 
         |         | Necessary to use winsound.
         |         | pip install pypiwin32 
--------------------------------------------------
Tkinter  |         | Constructs GUI. 
         |         | Preinstalled with Python 2
